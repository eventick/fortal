# Eventick's Bootstrap


### Set up & Install dependencies

```
$ npm install -g gulp
$ git clone https://github.com/eventick/lollapalooza.git
$ cd lollapalooza
$ npm install
```

### Server

```
$ gulp start
```

### Dist

```
$ gulp dist
```
